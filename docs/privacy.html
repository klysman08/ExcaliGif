Privacy Policy

Effective date: July 3, 2026

Excali Up enhances Excalidraw by playing user-added animated GIFs and rendering customizable motion effects on selected lines and arrows. This policy explains how the extension handles data.

Short version: Excali Up does not collect, transmit, sell, or share personal information. Canvas data and animation settings are processed locally in your browser.

Data handled by the extension

To provide its visible features, Excali Up locally accesses:

  • Excalidraw canvas elements, including selected lines, arrows, element identifiers, and geometry.
  • GIF image data added by the user to an Excalidraw canvas.
  • Animation preferences such as style, speed, direction, size, spacing, glow, and playback state.

This information is used only to render GIF playback and motion effects inside Excalidraw.

Data collection and transmission

The developer does not receive canvas content, GIF files, browsing history, usage analytics, personal information, or extension settings. Excali Up has no developer-operated data server, advertising service, analytics service, account system, or tracking technology.

If Excalidraw provides a GIF through a URL, Excali Up may request that GIF data solely so the browser can decode and display it. The data is processed locally and is not sent to the Excali Up developer.

Local storage

Excali Up stores extension preferences and per-element animation settings in browser local storage associated with Excalidraw. This data remains on the user's device. Users can remove it by clearing site data for excalidraw.com in Chrome.

Permissions

  • activeTab: used only when the user opens the popup to confirm the active tab is Excalidraw and exchange extension status and settings.
  • Host access to excalidraw.com: required to integrate with the Excalidraw canvas and render the requested animations. Excali Up does not run on other websites.

Remote code

Excali Up does not execute remotely hosted JavaScript or WebAssembly. All executable code is included in the extension package distributed through the Chrome Web Store.

Sharing, selling, and unrelated use

Excali Up does not sell or transfer user data to third parties. It does not use data for advertising, creditworthiness, lending, profiling, or any purpose unrelated to its single purpose.

Chrome Web Store Limited Use

Excali Up's handling of information complies with the Chrome Web Store User Data Policy, including the Limited Use requirements. Data accessed by the extension is used only to provide its user-facing GIF playback and line-animation features.

Changes to this policy

This policy may be updated when the extension's functionality or data practices change. The effective date above will be revised whenever an update is published.

Contact

Questions about this policy can be sent to klysman.rk@gmail.com or submitted through the ExcaliGif GitHub issue tracker.
